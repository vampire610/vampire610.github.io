### 测试本地与服务器的连接
``` bash
$ ssh -T git@(服务器/网站名)  
例如：  
github:  
$ ssh -T git@github.com  
gitee:   
$ ssh -T git@gitee.com  
coding:  
$ ssh -T git@e.coding.net
```

### Windows直接在gitbush升级git
``` bash
$ git update-git-for-windows
```

### git查看远程仓库地址
``` bash
$ git remote -v
```
